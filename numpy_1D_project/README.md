# NumPy 1D Mini Project

## 📌 Topic
Student Marks Analyzer using NumPy 1D Arrays

## 🚀 Features
- Generate random student marks
- Sorting using NumPy
- Calculate average, max, min
- Standard deviation
- Pass/Fail classification using vectorization

## 🧠 Concepts Used
- 1D Arrays
- Indexing & Slicing
- NumPy functions (mean, max, min, std)
- Boolean masking
- Vectorization

## ▶️ Run
python main.py
